## 👨🏻‍⚖️ MSA 실시간 가격 하락 경매 서버
---
msa로 구성한 실시간 가격 하락 경매 서비스입니다.
사용자들 간 경매 서버에 물품을 등록, 가격 정책 및 가격 할인 정책을 적용하여 시간에 따라 할인된 가격의 경매 상품을 구매, 구입 취소 등의 서비스를 제공합니다.
